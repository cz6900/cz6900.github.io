植物盐处理办法

发表于 2022-02-23 更新于 2022-04-25 分类于 科研教程

植物盐处理办法

1 配置好各浓度溶液

0，100，200，300，400mM 各1L水，可浇水同浓度层次植物8盆

2 长期处理

从花盆上部浇灌盐溶液，直至溶液渗出底部

之后再次浇灌盐溶液，至溶液渗出底部

反复2-3次，注意浇灌盐溶液时，应该在同大盆内的小盆都已渗出溶液时

最后应该倒出液体

3天后，加入从大盆底部浇灌纯水，吸透后，将水倒净

再3天后，加入盐溶液，照第一次做法即可。

3 短期处理

盐处理后，不排尽盐溶液，待到第二天直接进行测定性状或取样

4 注意

最好使用红砂和蛭石3:1配基质

另外红砂需要经过筛选，清洗，将盐分清楚

宜使用网袋，穴盘铺成一个平面为宜，晾干2天后，烘干，和蛭石混合，备用

深圳I/O笔记

1 监控摄像头

active为up档位，亮红灯

network为up档位，亮蓝灯

同时up，同时亮，同时down，同时熄。

mov 0 p0

slp 6

mov 100 p0

slp 6

———————>p0 active

mov 0 p0

slp 4

mov 100 p0

slp 2

mov 0 p0

slp 1

mov 100 p0

slp 1

——————–>p0 network

2 工厂加强控制信号

———————>p0 input

mov p0 acc
mul 2
mov acc p1
slp 1 #输出数据后必须保持1秒，否则波线不走

———————>p1 output

3 诊断脉冲发生器

———————>p0 input

loop:
teq p0 0

  • +slp 1
  • +jmp loop

  • -mov 100 p1

  • -slp 1

  • -mov 0 p1

  • -slp 1
    end:
    mov 0 p1

    ———————>p1 output

———————>p0 input

tgt p0 0

  • mov 0 acc
  • not
    mov acc p1
    slp 1

———————>p1 output

———————>p0 input

teq p0 100

+gen p1 1 1

-slp 1

———————>p1 output

4 发光标志

loop:

teq p0 100

+add 1

+mov acc x1

loop:

teq p1 100

+sub 2

+mov acc x1

end:

end:

-slp 1

说明书里没写，但邮件内容里有写的：
gen p0 2 3
等价于
mov 100 p0
slp 2
mov 0 p0
slp 3
另外，那段邮件还介绍了“@” 的用法，用来标识在句首的话，本句就只执行一次，用于初始化。

2 linux更改安装目录

可以使用cd命令将当前目录切换到指定的文件夹。cd是change directory的缩写。
命令格式：cd [目录名]
有几个符号作为目录名有特殊的含义：“/”代表根目录、“..”代表上一级目录、“~”代表HOME目录、“-”代表前一目录。
用法示例：切换当前目录到/var/log目录
$ cd /var/log

*** 3 mkdir命令创建目录***

1、直接用mkdir指令

2、mkdir -p ：递归创建目录，即使上级目录不存在，会按目录层级自动创建目录

mkdir -p xx/yy的好处就是一次可以创建多级文件夹，若xx文件夹不存在，则先创建xx文件夹，然后在xx文件夹下创建yy文件夹

1、创建 test01 、test02、test03 三个文件夹命令：

[root@localhost ~]# mkdir test01 test02 test03
1
2、使用 -p 参数来创建多级文件夹，例如： test01/test02/test03

[root@localhost ~]# mkdir -p test01/test02/test03
1
3、在多个文件夹里的每个文件夹里面需要创建一个tool文件夹的话，可以用如下命令实现：

[root@localhost ~]# mkdir {test01,test02,test013}/tool
