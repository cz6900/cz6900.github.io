深圳I/O笔记

1 监控摄像头

active为up档位，亮红灯

network为up档位，亮蓝灯

同时up，同时亮，同时down，同时熄。

mov 0 p0

slp 6

mov 100 p0

slp 6

———————>p0 active

mov 0 p0

slp 4

mov 100 p0

slp 2

mov 0 p0

slp 1

mov 100 p0

slp 1

——————–>p0 network

2 工厂加强控制信号

———————>p0 input

mov p0 acc
mul 2
mov acc p1
slp 1 #输出数据后必须保持1秒，否则波线不走

———————>p1 output

3 诊断脉冲发生器

———————>p0 input

loop:
teq p0 0

  • +slp 1
  • +jmp loop

  • -mov 100 p1

  • -slp 1

  • -mov 0 p1

  • -slp 1
    end:
    mov 0 p1

    ———————>p1 output

    ———————>p0 input

    tgt p0 0

  • mov 0 acc

  • not
    mov acc p1
    slp 1

———————>p1 output

———————>p0 input

teq p0 100

+gen p1 1 1

-slp 1

———————>p1 output

4 发光标志

loop:

teq p0 100

+add 1

+mov acc x1

loop:

teq p1 100

+sub 2

+mov acc x1

end:

end:

-slp 1

说明书里没写，但邮件内容里有写的：
gen p0 2 3
等价于
mov 100 p0
slp 2
mov 0 p0
slp 3
另外，那段邮件还介绍了“@” 的用法，用来标识在句首的话，本句就只执行一次，用于初始化。
